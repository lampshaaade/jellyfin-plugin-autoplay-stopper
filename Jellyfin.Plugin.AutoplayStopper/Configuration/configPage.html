🛑 AutoPlay Stopper

Ce plugin ajoute un bouton directement dans le lecteur vidéo Jellyfin pour désactiver la lecture automatique de la session en cours.
Idéal pour éviter qu'une série continue de jouer en cas d'endormissement.

Injecte le bouton AutoPlay dans le lecteur vidéo. Désactiver cette option retire le bouton sans désinstaller le plugin.
Si coché, chaque nouvel onglet démarrera avec l'autoplay bloqué. L'utilisateur pourra le réactiver manuellement depuis le lecteur.

Comment utiliser

  1. Activez le plugin et sauvegardez.
  2. Rechargez la page Jellyfin (Ctrl+F5).
  3. Lancez la lecture d'un épisode — le bouton Autoplay ON/OFF apparaît dans la barre du lecteur.
  4. Cliquez dessus pour bloquer l'autoplay. Le bouton devient rouge.
  5. L'état se remet à ON automatiquement à la fermeture de l'onglet.
